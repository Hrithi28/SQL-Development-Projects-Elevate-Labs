# 🗄️ SQL & Database Management Projects Collection

A collection of SQL and database management projects demonstrating database design, normalization, ETL processes, query optimization, reporting, and database automation using MySQL, PostgreSQL, and SQLite.

---

# 📌 Projects Included

## 1. Online Retail Sales Database Design

A normalized relational database designed for an e-commerce platform to efficiently manage products, customers, orders, payments, and sales reporting.

### ✨ Features

- Entity Relationship (ER) Diagram
- Normalized database schema (3NF)
- Primary & Foreign Key constraints
- DDL scripts for table creation
- Sample data population
- Complex JOIN queries
- Sales report views
- Optimized relational structure

### 🛠 Tech Stack

- MySQL / PostgreSQL
- dbdiagram.io
- SQL

---

## 2. Employee Management & Attendance Tracker

A complete employee management database that tracks attendance, departments, roles, and work hours with automation using SQL triggers and functions.

### ✨ Features

- Employee management
- Department & role management
- Attendance tracking
- Monthly attendance reports
- Late arrival detection
- SQL Functions
- SQL Triggers
- Timestamp automation
- Work hour calculation
- Aggregate reporting

### 🛠 Tech Stack

- PostgreSQL
- pgAdmin
- SQL

## 3. SQL ETL Pipeline Simulation
